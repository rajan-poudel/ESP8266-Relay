# ESP8266-Relay
This repository contains code for ESP8266(NodeMCU 1.0 Black) to control two relays from a web server hosted in local network.

# Circuit
In  ( Relay 1 ) ==> D1 <br>
VCC ( Relay 1 ) ==> 3V <br>
GND ( Relay 1 ) ==> G  <br><br>

In  ( Relay 2 ) ==> D2 <br>
VCC ( Relay 2 ) ==> 3V <br>
GND ( Relay 2 ) ==> G
